Functions and equivalence relations

Muruhan Rathinam

We assume familiarity with the concepts of sets, elements, subsets, equality of sets, and cartesian products of two sets. Also we assume familiarity with concepts and notations of element belonging to a set (∈), subsets (⊆), unions (∪), intersections (∩), relative complement (∖) and complement Ac of a set A (same as S∖A where S is known from the context). Moreover, familiarity with the concept of the Cartesian product A×B of two sets A and B is assumed.

1 Functions

Given nonempty sets A and B, a function (or map) f from A into B denoted by f:A→B is an assignment of an element of B to each element of A. The element of B assigned to an element a∈A by f is denoted by f⁢(a). The set A is the domain of the function f, the set B is the codomain of f.

Given two functions f,g:A→B (with same domain and codomain), we shall say f=g provided that for all x∈A, f⁢(x)=g⁢(x).

Given a function f:A→B and a subset E⊆A the direct image or simply the image of E under f is denoted by f⁢(E) and is the set

f⁢(E)={y∈B|∃x∈A⁢ s.t. ⁢f⁢(x)=y}.

Given a subset F⊆B, the inverse image of F under f denoted by f−1⁢(F) is defined by

f−1⁢(F)={x∈A|f⁢(x)∈F}.

The range of a function f:A→B is f⁢(A).

A function f:A→B is said to be injective or one-to-one provided that for all x,y∈A if f⁢(x)=f⁢(y) then x=y.

A function f:A→B is said to be surjective or onto provided that for all y∈B there exists x∈A such that y=f⁢(x). Equivalently, f is surjective if its range equals its codomain.

A function f:A→B is said to be bijective or a one-to-one correspondence provided it is both injective and surjective.

Given sets A,B and C, and functions f:A→B and g:B→C, the composition g∘f of g and f is the function g∘f:A→C defined by

(g∘f)⁢(x)=g⁢(f⁢(x))∀x∈A.
Proposition 1

Composition of functions is associative. That is, if f:A→B, g:B→C and h:C→D then

h∘(g∘f)=(h∘g)∘f.

Given a set S, the identity on S is the function idS:S→S defined by idS⁢(x)=x for all x∈S.

Given a set S and a subset T⊆S the inclusion function ι:T→S of T into S is defined by ι⁢(x)=x for all x∈T.

Let f:A→B be a function. A function g:B→A is said to be an inverse of f provided

g∘f=idA and ⁢f∘g=idB.

From the definition it follows that g is an inverse of f if and only if f is an inverse of g.

Proposition 2

If f:A→B has an inverse, then it is unique.

Proof.

Let g,h:B→A both be inverses of f. Then

g∘(f∘h)=g∘idB=g,

and

(g∘f)∘h=idA∘h=h.

The associativity of compositions implies g=h. ∎

When f:A→B has an inverse, its unique inverse is denoted by f−1. Do not confuse the inverse function notation f−1 with the inverse image notation f−1!

Proposition 3

Let f:A→B be a function. Then f has an inverse if and only if it is a bijection.

Proof.

Suppose f is a bijection. Define g:B→A as follows. Given y∈B arbitrary, since f is surjective, there exists x∈A such that f⁢(x)=y. Moreover, since f is injective, there is only one such x. Define g⁢(y)=x. Having defined g, we shall show that g is the inverse of f.

First we show that g∘f=idA. To that end, given arbitrary x∈A, we see that (g∘f)⁢(x)=g⁢(f⁢(x))=x by definition of g.

Next we show that f∘g=idB. To that end, let y∈B be arbitrary. Then there exists one and only one x∈A such that y=f⁢(x). By definition of g, x=g⁢(y). Hence, (f∘g)⁢(y)=f⁢(g⁢(y))=f⁢(x)=y.

To show the converse, suppose that f has an inverse, say g:B→A.

We shall first show that f is a surjection. To that end, let y∈B be arbitrary. Then since f∘g=idB, it follows that f⁢(g⁢(y))=y. Let x=g⁢(y)∈A. Thus there exists x∈A such that f⁢(x)=y, showing f to be surjective.

Next we shall show that f is injective. Suppose x,x′∈A are such that f⁢(x)=f⁢(x′). Then g⁢(f⁢(x))=g⁢(f⁢(x′)). Thus (g∘f)⁢(x)=(g∘f)⁢(x′) and hence idA⁢(x)=idA⁢(x′), which implies that x=x′. This shows that f is injective. ∎

Given a nonempty set A, a binary operation on A is a function f:A×A→A. The term magma is used to denote the pair (A,f) where A is a nonempty set and f is a binary operation on A. Magmas, that is, sets with a binary operation on them will play a central role in abstract algebra. Given a set A with a binary operation f, it is often common to write x⁢f⁢y in place of f⁢(x,y). As an example, consider the operation of addition + on the set of real numbers. We write x+y instead of +(x,y).

2 Equivalence relations

Given a pair of nonempty sets A and B, a binary relation R over A and B is a subset of A×B. If A=B we use the phrase binary relation on A. Given x∈A and y∈B, we often write x⁢R⁢y to indicate that (x,y)∈R.

Given a set S, a binary relation ∼ on S is said to be an equivalence relation provided

  1. 1.

    x∼x for all x∈S. Reflexive.

  2. 2.

    For all x,y∈S, x∼y if and only if y∼x. Symmetric.

  3. 3.

    For all x,y,z∈S, if x∼y and y∼z then x∼z. Transitive.

Let ∼ be an equivalence relation on a nonempty set S. Given an element x∈S, the equivalence class of x with respect to ∼, denoted by [x] is the subset of S defined by

[x]={y∈S|x∼y}.

A nonempty subset A⊆S is an equivalence class with respect to (S,∼) provided there exists x∈S such that A=[x].

Lemma 1

Let ∼ be an equivalence relation on a nonempty set S. Suppose A,B are equivalence classes. Then either A=B or A∩B=∅.

Proof.

Suppose A∩B is nonempty. Let x∈A∩B. We shall show that A=B.

Let y∈A be an arbitrary element. Since A is an equivalence class and x,y∈A, it follows that x∼y. Since x∈B, B is an equivalence class and x∼y it follows that y∈B. Thus A⊆B.

A similar argument shows B⊆A. ∎

Recall the notation 2S which stands for the power set of a set S which consists of every subset of S including S and ∅.

The set of all equivalence classes of (S,∼) is called the factor set of (S,∼). Note that the factor set is a subset of 2S and it is denoted by S/∼.

Given a nonempty set S, a subset P⊆2S is said to be a partition of S provided P satisfies the following conditions.

  1. 1.

    For each A∈P, A is nonempty.

  2. 2.

    If A,B∈P are distinct (meaning A≠B), then A∩B=∅.

  3. 3.

    The union of all elements of P is S:

    ⋃A∈PA=S.
Proposition 4

Let ∼ be an equivalence relation on a nonempty set S. Then S/∼ is a partition of S. Conversely if P is a partition of S, then there exists a unique equivalence relation ∼ on S such that P=S/∼.

Let S be a nonempty set and ∼ an equivalence relation on S. The function π:S→S/∼ defined by

π⁢(x)=[x]∀x∈S

is said to be the natural projection from S onto the factor set S/∼.

Equivalence relation associated with a map: Given f:S→T where S,T are nonempty sets, we can define an equivalence relation ∼f on S by setting x∼fy if and only if f⁢(x)=f⁢(y) for all x,y∈S.

Theorem 1

Let f:S→T be a function and suppose ∼f is the equivalence relation defined above. Then the function f¯:S/∼f→f⁢(S) defined by

f¯⁢([x])=f⁢(x)∀x∈S,

is a bijection.

Given f:S→T one may write

f=ι∘f¯∘π

where ι:f⁢(S)→T is the inclusion and π:S→S/∼f is the natural projection.
